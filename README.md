# 🌌 Star Wars - Guia de Trilogias

Este é um aplicativo mobile desenvolvido com **React Native** e **Expo**, que organiza os filmes da franquia Star Wars por ordem cronológica e suas respectivas trilogias. O projeto foi estilizado com foco em uma experiência temática, utilizando as cores clássicas da série (Amarelo e Preto).

## 🚀 Funcionalidades

- **Navegação Fluida**: Utiliza o React Navigation 7 para transições suaves entre as telas.
- **Identidade Visual**: Tema dark com cores contrastantes inspiradas na identidade visual original da franquia.
- **Informações Detalhadas**: Breve descrição sobre cada trilogia (Prequela, Original e Sequela) e fotos dos protagonistas.
- **Interface Responsiva**: Design adaptável para diferentes tamanhos de tela usando `ScrollView`.

As seguintes ferramentas foram fundamentais na construção deste projeto:

## 📦 Como rodar o projeto?

1. Certifique-se de ter o **Node.js** e o **npm** instalados.
2. Clone este repositório.
3. Instale as dependências:
   ```bash
   npm install
   ```
4. Inicie o servidor do Expo:
   ```bash
   npx expo start
   ```
5. Escaneie o QR Code com o aplicativo **Expo Go** no seu celular ou utilize um emulador Android/iOS.
